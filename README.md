# Image Community using Firebase


## 👋🏻 프로젝트 소개

혼자서 처음으로 만든 파이어베이스를 사용한 이미지 커뮤니티입니다! 👏🏻


### 👷🏻‍♂️ 주 기능

- 회원가입기능
- 이미지 업로드, 댓글 추가 및 수정


### 🔨 개발툴

-   Server: AWS EC2 (Ubuntu 20.04 LTS)
-   Language: Java Script
-   Framework/Library: React
-   View : Styled Components, Materail UI
